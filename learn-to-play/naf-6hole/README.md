# learn-to-play / Native American Flute (6-hole)

A starter songbook for the **A-tuned 6-hole Native American flute**
you built from [`flutes`](https://github.com/tonykoop/flutes) (the
parametric design table family). Generated by the
[`sheet-music`](https://github.com/tonykoop/sheet-music) skill.

This songbook is calibrated for the **A-tuned NAF** specifically —
fundamental A4, range A4–D6, A minor pentatonic. If your build is in a
different key (G, F#, D are also common in the registry), the same
tune.abc files transpose naturally; re-deposit with `--instrument
naf-6hole` after updating the registry row.

## Folder map

| Folder | Contents |
|---|---|
| `00-warmup-scales/` | Range walk, pentatonic intervals, and the single-grace ornament drill |
| `01-easy/` | Three slow, beginner-friendly PD tunes |
| `02-intermediate/` | Two brisker PD tunes with wider intervals |
| `03-original/` | *Pre-Dawn Field* — the Heifer Zephyr original written for this build |
| `fingering-charts.svg` | Combined fingering chart for every NAF pitch (A4–D6) |
| `songsheet.pdf` | Printable one-page summary of the whole songbook |

### 03-original

**Pre-Dawn Field** — composed for this build. AABA, 32 bars, ♩=60.
Written for the Heifer Zephyr origin moment: chasing aurora north of
Minneapolis, sitting up all night counting constellations, watching
cold fog settle on the field as the sun rose, breath flaring in the
chill like a heifer bison's in a Yellowstone morning. Read the full
story in [`03-original/pre-dawn-field/notes.md`](03-original/pre-dawn-field/notes.md).

## How to use the per-tune files

Every tune folder has multiple files. The canonical source is
`tune.abc` — everything else is generated from it.

| File | Purpose |
|---|---|
| `tune.abc` | Canonical notation in ABC format (plain text) |
| `tune.musicxml` | Notation for Ableton, Logic, MuseScore, Sibelius, etc. |
| `tune.mid` | Playable MIDI (uses GM preset 76 — Pan Flute) |
| `tune-fingering.svg` | Fingering chart, one diagram per distinct pitch |
| `tune.pdf` | Per-tune one-page printable songsheet |
| `notes.md` | Source / provenance, range notes, practice tip |
| `render-summary.json` | Which pipeline stages succeeded / were skipped |

> **Skipped stages on this generation:** `tune.ly` (LilyPond not
> installed locally), `tune.svg` (depends on LilyPond), and `tune.wav`
> (fluidsynth not installed). These are honest skips — install the
> binaries and re-run `scripts/render_pipeline.py` to fill them in.
> See `references/audio-rendering.md` in the `sheet-music` repo.

## Suggested practice path

1. Open the day with `00-warmup-scales/range-walk.abc`. Hold each note
   long enough to find a steady tone before moving up. Ascending,
   descending, twice through.
2. Run `intervals.abc` once. The 5ths and the octave are the wider
   embouchure stretches you'll need for the songbook.
3. Pick one tune from `01-easy/`. Open its `tune.mid` to hear the
   melody, then read its `notes.md` for the range check and practice
   tip.
4. Play the first phrase slowly with the per-tune fingering SVG open.
   Don't worry about tempo — find the breath first, the rhythm second.
5. When all three easy tunes feel comfortable, move to
   `02-intermediate/`. Simple Gifts and Twinkle Twinkle have wider
   intervals and brisker tempos.
6. Save `03-original/pre-dawn-field/` for a quiet hour. It's meant to
   be played slowly, alone.

## Re-generating

If you change a `tune.abc` (transposing, ornamenting, fixing a typo),
regenerate the derived files:

```bash
# from the sheet-music repo root:
python scripts/render_pipeline.py \
  --tune <tune.abc path> \
  --instrument naf-6hole \
  --out <output dir>
```
